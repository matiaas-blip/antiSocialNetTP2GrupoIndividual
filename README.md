# 🚀 UnaHur Anti-Social Net API

Backend de red social desarrollado con Node.js, Express y MongoDB.

---

## 🧠 Tecnologías

- Node.js
- Express
- MongoDB Atlas
- Mongoose
- Redis (cache)
- Swagger

---

## 📌 Funcionalidades

### 👤 Usuarios
- Crear usuario
- Listar usuarios
- Seguir usuarios

### 📝 Posts
- Crear posts con imágenes (URLs)
- Listar posts
- Feed personalizado

### 💬 Comentarios
- Crear comentarios
- Ocultar comentarios
- Filtrar por antigüedad (X meses)

### 🏷 Tags
- Crear tags
- Asociar a posts
